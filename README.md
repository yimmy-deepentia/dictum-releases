# Dictum — Descargas

**Dictado de voz local, privado y offline.** Escribe hablando, sin que tu voz salga de tu computadora.

## Descargar

Descarga la última versión desde la página de **[Releases](https://github.com/yimmy-deepentia/dictum-releases/releases/latest)**.

| Sistema | Archivo |
| --- | --- |
| **Windows** (10/11, x64) | `Dictum_0.1.0_x64-setup.exe` |
| **Windows** (ARM64) | `Dictum_0.1.0_arm64-setup.exe` |
| **macOS** (Apple Silicon) | `Dictum_0.1.0_aarch64.dmg` |
| **macOS** (Intel) | `Dictum_0.1.0_x64.dmg` |
| **Linux** (x64) | `.deb` · `.AppImage` · `.rpm` |
| **Linux** (ARM64) | `.deb` · `.AppImage` · `.rpm` |

> La versión de Windows x64 incluye el motor de IA local para pulir tu dictado sin conexión.

## Instalación (versión sin firmar)

Estos instaladores aún no están firmados con Apple/Microsoft, así que el sistema mostrará un aviso la primera vez:

- **Windows:** Si aparece *«Windows protegió tu PC»*, haz clic en *Más información* → *Ejecutar de todas formas*.
- **macOS:** La primera vez, clic derecho en Dictum → *Abrir* → *Abrir*. Solo la primera vez.
- **Linux:** Para el AppImage, dale permiso de ejecución (`chmod +x`) y ábrelo; o instala el `.deb`/`.rpm` según tu distribución.

---

Este repositorio aloja únicamente las descargas. Dictum es un proyecto de dictado de voz privado por diseño.
